Pocket Agent Sandbox

Local-first LangGraph learning project: graph mechanics, tool use, memory, streaming, HITL, persistence, Server/Studio, and provider experiments.

Objective

Build and verify a compact LangGraph foundations project that explains core primitives by running them locally, not by hiding them behind a product shell.

What It Can Do

  • Run a hand-built ReAct graph with actual local tools.
  • Keep thread memory through a checkpointer.
  • Stream graph progress into this page.
  • Pause before a side effect and resume from browser approval.
  • Browse local project reference docs without model recall.
  • Talk to LM Studio or user-supplied provider keys.
